# Noxtis releases

Downloads and auto-update artifacts for [Noxtis](https://github.com/NotPandaByte). Grab the latest from the [Releases page](https://github.com/NotPandaByte/noxtis-releases/releases/latest).

Linux one-liner:

```sh
curl -fsSL https://raw.githubusercontent.com/NotPandaByte/noxtis-releases/main/install.sh | bash
```
